Fixed ~ for tables

"Fixed"(fx) is a light weight javascript plug-in to cause a header and footer elements to stay fixed.
Also included are a few preset styling themes that can be applied to a table.

How to Use

  • Include JQuery: <script src="http://code.jquery.com/jquery-1.10.2.min.js" type="text/javascript"></script>
  • Include fx.js: <script src="path/to/fx.js" type="text/javascript"></script>
  • Include fx.css: <link href="path/to/fx.css" rel="stylesheet" type="text/css" >
  • Add in a table that has all these elements:
    <table>
     <thead>
      <tr>
       <th></th>
      </tr>
     </thead>
     <tbody>
      <tr>
       <td></td>
      </tr>
     </tbody>
     <tfoot>
      <tr>
       <th></th>
      </tr>
     </tfoot>
    </table>
  • Insert onto the table tag: class="fx"
  • Insert into that class a style of your choice (nsb, rtg, fbr): class="fx nsb"
